> ## Instalação :computer:

1. git clone https://github.com/corebiz-global/babadotop.git
2. cd babadotop
3. git flow init
    - Aperte a tecla _enter_ para todos os _prefixes_ **menos** \*_Version tag prefix_\*, nesse escreva a letra **v** e aperte _enter_
4. yarn install
